Editing¶

The primary purpose of rstweaver is to be able to edit source files from the rst document.

Naming a block¶

.. cpp:: main.cpp done
    :name: includes

    #include <iostream>
    #include <cstdlib>


     main.cpp 

  1  #include <iostream>
  2  #include <cstdlib>
  3

Interactive¶


.. icpp::

    int x = 5
    std::cout << x






c++> int x = 5


c++> std::cout << x
5





Which isn’t terribly handy unless you want to use some code you’ve written:


.. cpp:: collatz.hpp

    int collatz(int n);

.. cpp:: collatz.cpp

    int collatz(int n) {
        int k = n;
        int count = 0;

        while (k != 1) {
            if (k % 2 == 0) k = k/2;
            else k = 3*k + 1;

            count++;
        }

        return count;
    }

.. icpp:: collatz.cpp collatz.hpp

    std::cout << collatz(1)
    std::cout << collatz(2)
    std::cout << collatz(3)
    std::cout << collatz(4)
    std::cout << collatz(5)

    {-# LANGUAGE OverlappingInstances #-}
    {-# LANGUAGE FlexibleInstances #-}
    {-# LANGUAGE ScopedTypeVariables #-}

    class CountArgs a where
        countArgs :: a -> Int

    instance (CountArgs b) => CountArgs (a -> b) where
        countArgs _ = (1::Int) + (countArgs (undefined :: b))

    instance CountArgs a where
        countArgs _ = (0::Int)

    main = do
        print $ countArgs (:)
